# Uduki 

markdownで記述する日々のメモ帳です。日付が代わると自動的に最新へ書き換わります。
何かメモしておきたいけどファイル名つけるのも面倒だしBlogに書くほどでもない。
もしくは、公に書けないけどメモっておきたい等の目的に適しております。
